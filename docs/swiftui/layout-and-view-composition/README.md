# Layout and View Composition

## Preparation Paths

### Rapid Review

Study [Layout Proposal and Response](layout-proposal-and-response/README.md), then
[Stacks, Grids, and Safe Areas](stacks-grids-and-safe-areas/README.md). Add
[Alignment, Geometry, and Preferences](alignment-geometry-and-preferences/README.md)
when the interview includes layout debugging or reusable components.

### Standard Preparation

1. [Layout Proposal and Response](layout-proposal-and-response/README.md)
2. [Stacks, Grids, and Safe Areas](stacks-grids-and-safe-areas/README.md)
3. [Alignment, Geometry, and Preferences](alignment-geometry-and-preferences/README.md)
4. [Custom Layouts and Adaptive Composition](custom-layouts-and-adaptive-composition/README.md)

### Role-Specific Depth

For UI-platform, design-system, or layout-heavy roles, rehearse the custom `Layout`,
preference aggregation, adaptive component contracts, and performance trade-offs.
