# Requirements, Conformance, and Synthesis

> A protocol is a behavioral contract; compiling a witness is necessary but semantic laws and ownership expectations determine substitutability.

## Quick Recall

- Requirements describe capability, not storage.
- A conformance must satisfy semantic rules as well as type signatures.
- Synthesis is useful only when generated behavior matches domain meaning.
