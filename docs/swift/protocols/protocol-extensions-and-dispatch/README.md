# Protocol Extensions and Dispatch

> Default implementations can satisfy requirements dynamically through witnesses; extension-only members are selected from static type and constraints.

## Quick Recall

- A default implementation can satisfy a declared requirement.
- A requirement call uses the selected conformance witness.
- An extension-only member is chosen from static type information.
