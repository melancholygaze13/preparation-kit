# Strings and Characters

## Scope

Swift text as Unicode-correct value semantics: extended grapheme clusters,
encoding views, canonical equivalence, native indices, mutation, substring
lifetime, and conversion at Foundation and protocol boundaries.

Localization system design and user-facing formatting belong to Apple frameworks.
General collection range behavior belongs to Basic Operators.
