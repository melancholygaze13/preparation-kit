# Language Basics

## Scope

The foundational rules Swift uses to represent values, express absence, enforce
types, report failures, and validate program assumptions. The organization
follows the Swift book's “The Basics” chapter, while each concept emphasizes the
decisions and failure modes relevant to experienced iOS engineers.
