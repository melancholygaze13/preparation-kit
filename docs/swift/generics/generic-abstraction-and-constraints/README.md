# Generic Abstraction and Constraints

> Generics share one implementation across concrete types while constraints state the
> operations and semantic contracts that implementation may rely on.

## Quick Recall

- A generic parameter preserves concrete type information.
- Constraints state which operations generic code may use.
- Add a constraint only when the implementation or contract needs it.
