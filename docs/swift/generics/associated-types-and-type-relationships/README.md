# Associated Types and Type Relationships

> An associated type is a protocol placeholder chosen by each conformance; constraints
> connect it to other types so generic code can prove operations are valid.

## Quick Recall

- Each conformance chooses its associated type.
- Same-type constraints connect related generic values.
- Use an associated type when the relationship belongs to the conformer.
